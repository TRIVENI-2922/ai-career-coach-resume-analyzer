# AI Career Coach & Resume Analyzer

An AI-powered Career Assistant built using n8n and Google Gemini AI.

## Features

- Resume Analysis
- ATS Score Estimation
- Career Guidance
- Personalized Learning Roadmaps
- Mock Interview Preparation
- Conversation Memory

## Tech Stack

- n8n
- Google Gemini AI
- AI Agent
- Simple Memory

## Workflow

User → Chat Trigger → AI Agent → Gemini AI → Memory → Response
